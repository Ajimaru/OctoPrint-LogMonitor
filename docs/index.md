# OctoPrint Log Monitor

Live log streaming and search for OctoPrint, with severity alerts in the navbar and sidebar.

## What you get

- Real-time log streaming
- Full-text search with pagination and severity filters
- Navbar and sidebar alert badges
- Log download and alert history
