# Contributing

Keep changes small, testable, and focused.

## Before opening a PR

- Run the test suite locally
- Update docs when behavior changes
- Add a changelog entry when relevant
- Keep unrelated files out of the diff

## Local setup

- Python 3.9+
- A local OctoPrint environment for manual testing

For full contribution details, see the repository [CONTRIBUTING.md](https://github.com/Ajimaru/OctoPrint-LogMonitor/blob/main/CONTRIBUTING.md).
